\# Medical Device Risk Assessment Tool



A software-based Medical Device Risk Management application inspired by \*\*ISO 14971 principles\*\* and \*\*FMEA methodology\*\* for identifying hazards, evaluating risks, and documenting mitigation strategies.



\## Project Overview



Medical device companies must identify and control risks throughout the product lifecycle. This tool provides a simple digital platform to support risk analysis activities by calculating risk scores, managing hazards, performing FMEA analysis, and generating risk assessment reports.



\## Features



\- User authentication system

\- Hazard identification and documentation

\- Risk scoring based on:

&#x20; - Severity

&#x20; - Occurrence

&#x20; - Detectability

\- Automated risk classification

\- FMEA (Failure Mode and Effects Analysis) module

\- Risk mitigation tracking

\- PDF risk assessment report generation

\- Database management for storing assessments



\## Technologies Used



\- Python

\- Streamlit

\- SQLite Database

\- ReportLab (PDF Generation)

\- Git \& GitHub



\## Medical Device Standards Reference



This project is inspired by:



\- ISO 14971: Medical Devices — Application of Risk Management to Medical Devices

\- FMEA principles used in medical device quality engineering
